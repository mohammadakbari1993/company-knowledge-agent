# Project Brief

## Project Name

Mini Company Knowledge Bot

## Goal

Build a simple web application that allows users to ask questions about a company's policies, product information, and support procedures.

## MVP

- User enters a question
- System searches company documents
- System returns the most relevant answer

## Documents

- FAQ
- Leave Policy
- Product Guide

## Non Goals

- Authentication
- User accounts
- External databases
- Complex RAG pipelines
- Vector databases

## Success Criteria

A user can ask questions about company information and receive answers based on the provided documentation.
